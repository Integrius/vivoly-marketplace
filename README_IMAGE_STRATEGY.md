## Brand Consistency

### Color Palette (Emerald Fresh)

**Primary Colors:**
- Emerald Green: `#10B981` / `hsl(160, 84%, 39%)`
- Teal: `#14B8A6` / `hsl(172, 83%, 41%)`

**Usage in Images:**
- AI-generated images: Include colors in prompt
- Stock photos: Add 10-20% emerald overlay
- Icons: Use emerald as primary color
- Illustrations: Emerald primary, teal accent

### Visual Style Guide

**Photography:**
- Modern, clean, professional
- Bright natural lighting
- Tech/business focus
- Avoid: Overly staged, dated stock

**Illustrations:**
- Flat design or subtle gradients
- Emerald/teal color scheme
- Clean lines, not overly complex
- Modern, tech-focused aesthetic

**Icons:**
- Consistent style (outline or filled)
- Medium stroke weight (2px)
- Emerald color or neutral gray
- 24x24px base size

## Frequently Asked Questions

### General Questions

**Q: Do I need all 23 images?**
A: No. Start with 5-6 critical images (Quick Start). Add more over time.

**Q: What if I have zero budget?**
A: Use free tools: Leonardo.ai (150 free credits/day), Unsplash, Flaticon, Lucide React.

**Q: How long will this take?**
A: Quick Start (6 images): 2-3 hours. Full implementation (23 images): 30-35 hours spread over 4 weeks.

**Q: Do images slow down my website?**
A: Not if optimized correctly. Follow the performance guidelines (WebP, lazy loading, compression).

### Technical Questions

**Q: What image format should I use?**
A: WebP for photos/illustrations, SVG for icons, JPG for Open Graph images.

**Q: What size should my images be?**
A: Varies by location. See IMAGE_DIRECTORY_STRUCTURE.md for exact specifications.

**Q: How do I prevent layout shift?**
A: Always specify width/height or use `fill` with parent container having fixed height.

**Q: Should I use priority or lazy loading?**
A: Use `priority` for above-the-fold images (hero), lazy load everything else (default).

### Content Questions

**Q: Where can I get free images?**
A: Unsplash, Pexels, Pixabay for photos. Leonardo.ai for AI-generated. Flaticon for icons.

**Q: Can I use AI-generated images commercially?**
A: Yes, Leonardo.ai and most AI tools allow commercial use. Check terms for each tool.

**Q: What if I don't like the AI results?**
A: Generate 4 variations, pick the best. Or use stock photos with brand color overlay.

**Q: Do I need to hire a designer?**
A: No. Follow the prompts and guides provided. Leonardo.ai produces professional results.
