## Technical Settings for Each Tool

### Leonardo.ai Settings

**Best Settings:**
- Model: Leonardo Diffusion XL
- Dimensions: Custom (match required size)
- Photo Real: OFF (for illustrations), ON (for photography)
- Alchemy: ON (better quality)
- Prompt Magic: Level 2
- Number of images: 4 (generate multiple, pick best)

### Midjourney Parameters

**Recommended Parameters:**
- `--ar X:Y` - Aspect ratio (e.g., --ar 16:9)
- `--v 6` - Use version 6 (latest)
- `--style raw` - More literal interpretation
- `--q 2` - Highest quality (costs 2x tokens)
- `--no text, watermark` - Exclude unwanted elements

**Example Full Command:**
```
/imagine abstract neural network, emerald green color --ar 16:9 --v 6 --style raw --q 2 --no text
```

### DALL-E 3 Tips

**Best Practices:**
- Be very specific about colors (use hex codes)
- Mention "professional", "high quality", "realistic"
- Specify exact dimensions in the prompt
- Request "no text" if you don't want text
- Use natural language (DALL-E 3 understands context well)

### Stable Diffusion Settings

**Recommended Settings:**
- Model: SD XL 1.0 or SDXL Turbo
- Sampler: DPM++ 2M Karras
- Steps: 30-50
- CFG Scale: 7-9
- Size: Match required dimensions
- Negative prompt: "text, watermark, signature, blurry, low quality"

## Free Alternative: Stock Photos + Editing

If AI generation isn't working:

### Stock Photo Sites
1. **Unsplash** (https://unsplash.com)
2. **Pexels** (https://www.pexels.com)
3. **Pixabay** (https://pixabay.com)

### Editing Process
1. Download high-res stock photo
2. Open in Figma or Photopea (free Photoshop alternative)
3. Add emerald green overlay at 10-20% opacity
4. Add gradient overlay (emerald to teal)
5. Adjust brightness/contrast
6. Export as WebP

### Figma Quick Edit Steps
```
1. Import image to Figma
2. Add rectangle layer over image
3. Fill: Linear gradient (emerald #10B981 to teal #14B8A6)
4. Set opacity: 15%
5. Set blend mode: Multiply or Overlay
6. Export as PNG/WebP
```

## Quality Control

### Before Using Any Generated Image

Check:
- [ ] Resolution is high enough (no pixelation)
- [ ] Colors match brand (emerald/teal)
- [ ] No AI artifacts (weird details, distorted faces)
- [ ] No unwanted text or watermarks
- [ ] Appropriate for professional business context
- [ ] Looks good at mobile sizes too

### If Image Needs Improvement

**Upscaling:**
- Use Upscayl (free, open-source): https://upscayl.github.io
- Or Topaz Gigapixel AI

**Color Correction:**
- Use Figma/Photopea to adjust hue/saturation
- Add emerald green color grading if needed

**Touch-ups:**
- Remove unwanted elements with Photopea (free)
- Or use Cleanup.pictures (free AI removal)
